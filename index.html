程序入口文件

  • 一个文件中只能有一个 public class，且这个 class 的名字必须要与文件名一样
  • public class 中的 main 函数是程序的入口

标识符

  • 命名规则
    • 标识符可以包含英文字母、数字、_、$符号
    • 标识符不能以数字开头
    • 标识符不能是关键字
  • 命名规范
    • 类名：首字母大写，后面每个单词首字母大写（大驼峰式）
    • 方法名、变量名：首字母小写，后面每个单词首字母大写（小驼峰式）
